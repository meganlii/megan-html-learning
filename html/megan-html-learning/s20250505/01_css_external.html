個人背景與技術熱情

我是一位熱衷於技術挑戰與系統架構最佳化的後端工程師，擁有五年以上開發高效能、可擴展後端系統的經驗。我畢業於國立臺灣大學資訊工程學系，期間奠定了紮實的程式設計與演算法基礎，並參與多項專案，培養了團隊合作與問題解決能力。畢業後，我投入軟體產業，專注於後端開發，熟悉多種程式語言與框架，如 Python、Java、Node.js 以及 Spring Boot，並具備設計 RESTful API、GraphQL 與微服務架構的實戰經驗。

新創公司實戰經驗

在過去的工作中，我曾於一家新創公司擔任後端工程師，負責設計與維護高流量的線上交易平台。我成功優化資料庫查詢效率，將 API 響應時間縮短 30%，並導入 Redis 快取機制，提升系統吞吐量。此外，我參與了從 monolith 架構轉型至微服務的過程，確保服務間的高可用性與資料一致性。在另一家大型科技公司，我領導了一個五人團隊，開發雲端原生應用，整合 AWS 服務如 Lambda 與 DynamoDB，並實作 CI/CD 流程，大幅縮短部署週期。

大型科技公司領導經驗

我對技術保持高度熱情，持續學習新技術，如容器化（Docker、Kubernetes）與事件驅動架構，並積極參與技術社群，分享經驗。我擅長分析複雜問題、設計穩健解決方案，並注重程式碼品質與可維護性。在工作中，我重視與前端、DevOps 及產品團隊的協作，確保產品能滿足用戶需求並維持高穩定性。
